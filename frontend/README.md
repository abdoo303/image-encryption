# Hyperchaos Cryptography Frontend

This is the React frontend for the Hyperchaotic Cryptography System.

## Getting Started

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm start
```

The app will run on [http://localhost:3000](http://localhost:3000)

### Backend Connection

Make sure the Flask backend is running on port 5001:

```bash
cd ../backend
python server.py
```

## Features

- Initialize cryptographic system with custom seed
- Upload and encrypt images using hyperchaotic systems
- Decrypt encrypted images
- Visualize phase space trajectories
- View system parameters and encryption keys
